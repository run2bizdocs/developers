    Load Webservice Operations¶

    Description

    The objective is to load the scripts of Webservice operations in the language you have selected.

    Procedure¶

    1. Access the functionality through the main menu System > Database > Scripts > Upload Webservices Script;

    2. Select the language you want;

    3. Click on "Load".

    What to do next¶

    After load the webservice script, verify the language load through the webservice operation.
