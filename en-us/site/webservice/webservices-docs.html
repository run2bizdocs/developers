    Webservices 4biz Docs¶

    This section is intended to describe the communication structure REST, established between applications and the back-end server.

    Warning

    "4biz_URL": URL unalterable prefix, so that you can access the services made available to the mobile applications.
    ALL API THAT REQUIRES A “sessionID” WILL NEED TO BE IN A SESSION PROVIDED BY THE “Login” API.

    Login¶

    User Authentication Service

    GET

    <url_base>/cit-portal-web/rest/usuario/getToken
