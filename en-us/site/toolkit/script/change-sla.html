    Change SLA via Rhino script¶

    This functionality is intended to change the SLA through Rhino script.

    Procedure¶

    1. Access the main menu System > Batch Processing;

    2. Click on "New";

    3. Complete the fields available, being:

      • type: RhinoScript;

      NOTE

      Cron expression: it defines the routine execution time and its content, where it'll be described the context of the routine that will be executed in the tool.

      • content: see script in the next section.

    4. Click on "Save".

    Rhino Script¶
